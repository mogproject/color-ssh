Dependencies
------------

* Python: 2.6 / 2.7 / 3.2 / 3.3 / 3.4 / 3.5

------------
Installation
------------

* ``pip`` command may need ``sudo``

+-------------------------+---------------------------------------+
| Operation               | Command                               |
+=========================+=======================================+
| Install                 |``pip install color-ssh``              |
+-------------------------+---------------------------------------+
| Upgrade                 |``pip install --upgrade color-ssh``    |
+-------------------------+---------------------------------------+
| Uninstall               |``pip uninstall color-ssh``            |
+-------------------------+---------------------------------------+
| Check installed version |``color-ssh --version``                |
+-------------------------+---------------------------------------+
| Help                    |``color-ssh -h``                       |
+-------------------------+---------------------------------------+
